Learning to Code

Apart from a week-long day camp at the age of twelve, I had very limited experience with web development up until this past May. While discussing my open schedule with a friend, I was directed to this article, which provides numerous links for learning to code. Of the resources listed, all the "Beginner" links were helpful in some way. Below are my favourites.

Dash was the most helpful, providing tutorials on creating web content using HTML, CSS and JavaScript. Code can be entered on the left and the page is updated in real-time on the right, so you can see what effect your changes make.
W3 Schools is a full guide to coding everything from links to colours for a number of different languages. This is an excellent resource to learn the ins and outs of developing.
