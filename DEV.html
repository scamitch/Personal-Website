Learning to Code

Apart from a week-long day camp at the age of twelve, I had very limited experience with web development up until this past May. While discussing my open schedule with a friend, I was directed to this article, which provides numerous links for learning to code. Of the resources listed, all the "Beginner" links were helpful in some way. Below are details on the sites I've used.

Dash was the most helpful, providing tutorials on creating web content using HTML, CSS and JavaScript. Code can be entered on the left and the page is updated in real-time on the right, so you can see what effect your changes make. I completed each of the tasks ("projects"), but can refer back to the site easily for revision or help with a certain action.

W3 Schools is a full guide to coding everything from links to colours for a number of different languages. This is an excellent resource to learn the ins and outs of developing.

Code Academy is similar to Dash, where there are various tasks to complete. Although I found Dash more user-friendly, Code Academy has a much larger collection of assignments available, and I am still working through it. Code Academy includes lessons on HTML, CSS, jQueury, JavaScript, Python, and Ruby.

Creating this Website

This website was coded on Sublime Text, a free text editor that's great for the new developer or someone with more experience. In my case the most helpful feature is the colour-coded syntax of different tags: this makes it easier to keep track of the page and the various content, however, there are many features I haven't touched.

The full website was coded from scratch, but inspiration was drawn from the Dash projects.
